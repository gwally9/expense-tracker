## `Simple Expense Tracking we-base app`


## SETUP
`python3 -m venv expenses`
`source expenses/bin/activate`
`pip install flask`


### Build you templates   - defaults are included here
```
your-project/
├── app.py (the main file above)
└── templates/
    ├── base.html
    ├── index.html
    ├── add_expense.html
    ├── expenses.html
    └── analytics.html
```

# RUN

`python app.py`
